# DeskGitLight
A PCB in the shape of the GitHub cat that lights up more as it becomes darker!
## BOM:
- 1x Battery Holder
- 2x Potentiometer
- 2x LED
- 1x Photoresistor
- 2x NPN Transistor
- 2x 910 ohm resistor
- 2x 220 Ohm resistor
